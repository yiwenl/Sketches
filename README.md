# Sketches

Personal experiments—mostly WebGL sketches by [Yi-Wen Lin](https://wensday.co/).

**Live site:** [https://yiwenl.github.io/Sketches/](https://yiwenl.github.io/Sketches/)

Many pieces are not heavily optimised, so first load can take a moment. Some sketches may not behave the same on every device or browser; this repo is mainly a playground for trying ideas.
